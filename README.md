This is my professional portfolio website, showcasing my skills, projects, and experience. Built with a focus on performance, clean code, and user experience.

---

## 🌟 Key Features

### 🌍 Multi-language Support
Integrated **Vue I18n** for seamless switching between English and Russian. The user's language preference is automatically saved in `localStorage`, ensuring a consistent experience on return visits.

### 📱 Fully Responsive Design
Using **Tailwind CSS**, I've ensured that the layout adapts perfectly to any screen size—from small mobile devices to large 4K monitors.

### ⚡ Blazing Fast Performance
Leveraging **Vite** for optimized asset bundling and **Vue 3's Composition API** for efficient reactivity and small bundle sizes.

### 🎨 Clean & Minimal UI
A modern interface focusing on content readability, smooth transitions, and professional aesthetics.

## 📁 Project Directory Structure

```

portfolio/
├── public/              # Static assets (robots.txt, favicon)
├── src/
│   ├── assets/          # Global styles (Tailwind/SCSS) and images
│   ├── components/      # Atomic UI components (Header, Hero, Cards)
│   ├── locales/         # Translation JSON files (EN/RU)
│   ├── router/          # Vue Router configuration
│   ├── views/           # Main page layouts (Home.vue, About.vue)
│   ├── App.vue          # Root component
│   └── main.ts          # Application entry point & plugin config
├── tailwind.config.js   # Custom theme & utility configuration
├── tsconfig.json        # TypeScript compiler rules
└── vite.config.ts       # Vite build & proxy settings

```
